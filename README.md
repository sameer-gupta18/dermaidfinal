DermaAid is an all-in-one skincare and dermatology web application built with React. It provides users with tools to analyze skincare products, detect potential skin conditions, generate personalized skincare routines, and locate dermatologists nearby.  

---

## Features

1. *Ingredients Analysis*  
   - Scan or input a skincare product's ingredients list.  
   - Detect potentially harmful ingredients using OpenAI's API.  
   - Receive user-friendly explanations for each ingredient.  

2. *Melanoma Detection*  
   - Upload an image of a mole or skin lesion.  
   - Analyze the image using a custom-built Convolutional Neural Network (CNN).  
   - Get a preliminary assessment of potential melanoma.  
   > âš  For informational purposes only. Not a substitute for professional medical advice.  

3. *Personalized Skincare Routine*  
   - Answer a few questions about your skin type and goals.  
   - Receive a custom skincare routine generated by OpenAI's API.  

4. *Nearby Dermatologists*  
   - Detect and display dermatologists near your location.  
   - Integrated with Google Maps API for interactive map viewing.  

---

## Tech Stack

- *Frontend:* React, JavaScript, CSS  
- *Backend & AI:* OpenAI API, Custom CNN (TensorFlow/PyTorch)  
- *APIs:* Google Maps API for location services  
- *Other Tools:* Axios (API calls), React Router (navigation)
